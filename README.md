# Merge Film Festival

Website for the **Merge Film Festival (MFF)**, Switzerland's first film festival dedicated to AI.

## Run locally

This is a static website — no build step needed.

```bash
# from this folder
python3 -m http.server 8080
# then open http://localhost:8080
```

Or just open `index.html` in your browser.

## Structure

- `index.html` — home
- `assets/css/` — styles · `assets/js/` — scripts (carousels, etc.)
- `assets/img/` — films, events and UI images · `assets/video/` — hero videos

## Tech

Plain HTML, CSS and JavaScript.
